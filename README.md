## **SECTION A**

### **Question 1:**  
**What is the unit for measuring the output quality of a printer? Discuss some basic operations of a computer system.**  

**Answer:**  
The unit for measuring printer output quality is **DPI (Dots Per Inch)**. DPI indicates the resolution or clarity of the printed image.  

**Basic Operations of a Computer System:**  
1. **Input:** The process of entering data into the system.  
2. **Processing:** The CPU processes the input data based on instructions.  
3. **Storage:** Saving data permanently or temporarily.  
4. **Output:** Displaying the processed data to the user.  
5. **Control:** Managing the functioning of all operations.  

**Detailed Explanation (Hinglish):**  
- **Printer Output Quality (DPI):** Printer ki clarity ya resolution ko measure karne ke liye DPI ka use hota hai. Jitni zyada DPI, utni achhi quality.  

**Basic Operations:**  
1. **Input:** Jaise keyboard ya mouse se data dena.  
2. **Processing:** CPU data ko process karta hai.  
3. **Storage:** Data ko save karna ya temporary memory mein rakhna.  
4. **Output:** Processed data ko screen ya printout ke form mein dikhana.  
5. **Control:** Computer ke saare tasks ko manage karna.

### **Question 2:**  
**How is volatile memory different from non-volatile memory? Give examples of each.**  

**Answer:**  
- **Volatile Memory:**  
  - Data is lost when power is turned off.  
  - Example: RAM (Random Access Memory).  

- **Non-Volatile Memory:**  
  - Data is retained even when power is off.  
  - Example: Hard Disk, SSD, ROM.  

**Detailed Explanation (Hinglish):**  
- **Volatile Memory:**  
  - Yeh memory tab tak kaam karti hai jab tak power on ho. Agar power off hui, toh data erase ho jata hai. Example: RAM.  
- **Non-Volatile Memory:**  
  - Isme data hamesha safe rehta hai, chahe power off ho jaye. Example: Hard Disk aur ROM.

### **Question 3:**  
**What do you understand by ASCII code? Write the full form of ASCII code.**  

**Answer:**  
- **ASCII Code:** ASCII (American Standard Code for Information Interchange) is a character encoding standard used for representing text in computers and other devices.  
- It uses 7 or 8 bits to represent characters, including letters, numbers, and symbols.  

**Detailed Explanation (Hinglish):**  
- **ASCII Code:** ASCII ek encoding standard hai jo computers ko text samajhne mein madad karta hai. Har character ka ek unique binary number hota hai.  
- Example: Letter 'A' ka ASCII code **65** hota hai.

### **Question 5:**  
**Explain the following terms:**  
(i) Flowchart  
(ii) Algorithm  
(iii) Compiler and Interpreter  

**Answer:**  
(i) **Flowchart:** A flowchart is a diagrammatic representation of a process or algorithm using symbols.  

(ii) **Algorithm:** An algorithm is a step-by-step procedure to solve a problem or perform a task.  

(iii) **Compiler and Interpreter:**  
- **Compiler:** Translates the entire high-level code into machine language at once.  
- **Interpreter:** Translates and executes high-level code line by line.  

**Detailed Explanation (Hinglish):**  
- **Flowchart:** Yeh ek diagram hai jo kisi bhi process ya algorithm ko symbols ke through dikhata hai.  
- **Algorithm:** Problem solve karne ke liye step-by-step instructions ka set hota hai. Example: Cooking recipe.  
- **Compiler aur Interpreter:**  
  - Compiler code ko ek saath translate karta hai.  
  - Interpreter line-by-line code execute karta hai.

### **Question 6:**  
**Draw and explain the function block diagram of a computer. Explain the generations of computers in detail.**  

**Answer:**  
**Function Block Diagram:**  
The block diagram of a computer includes the following components:  
1. **Input Unit**  
2. **Output Unit**  
3. **Storage Unit**  
4. **Processing Unit (ALU + Control Unit)**  

**Generations of Computers:**  
1. **First Generation (1940–1956):** Used vacuum tubes; slow and large.  
2. **Second Generation (1956–1963):** Used transistors; faster and smaller.  
3. **Third Generation (1964–1971):** Used integrated circuits; more efficient.  
4. **Fourth Generation (1971–Present):** Used microprocessors; high performance.  
5. **Fifth Generation (Present and Beyond):** Focus on AI and quantum computing.  

**Detailed Explanation (Hinglish):**  
- **Block Diagram:** Yeh ek diagram hai jo computer ke main components ko dikhata hai, jaise input, output, memory, aur CPU.  

- **Generations of Computers:**  
  - **First Generation:** Vacuum tubes ka use hota tha, size bada aur speed slow hoti thi.  
  - **Second Generation:** Transistors aaye, system zyada reliable aur fast ban gaya.  
  - **Third Generation:** Integrated circuits ka use shuru hua, size chhota aur speed badh gayi.  
  - **Fourth Generation:** Microprocessors aaye, jisme poora CPU ek chip par hota hai.  
  - **Fifth Generation:** Artificial Intelligence aur Quantum Computing par focus ho raha

### **Question 2:**  
**Explain the basic components of a CPU with a block diagram.**  

**Answer:**  
**Components of CPU:**  
1. **Arithmetic and Logic Unit (ALU):** Performs arithmetic (addition, subtraction) and logical (AND, OR) operations.  
2. **Control Unit (CU):** Directs the operations of the computer by coordinating between hardware and software.  
3. **Registers:** Small memory units for temporary data storage during processing.  

**Block Diagram:**  
- **Input Unit** → **Control Unit + ALU** → **Memory Unit** → **Output Unit**  

**Detailed Explanation (Hinglish):**  
- **CPU Components:**  
  - ALU arithmetic aur logic operations karta hai.  
  - Control Unit hardware aur software ko synchronize karta hai.  
  - Registers temporary data ko hold karte hain.

### **Question 3:**  
**Explain the following:**  
(i) Registers and types  
(ii) DOS  
(iii) MS Word and MS Excel: Features and limitations  

**Answer:**  
(i) **Registers and Types:** Registers are small, high-speed memory units in the CPU.  
- **Types:**  
  1. **Instruction Register:** Holds the current instruction being executed.  
  2. **Program Counter:** Tracks the address of the next instruction.  
  3. **Accumulator:** Stores results of arithmetic/logic operations.  

(ii) **DOS (Disk Operating System):**  
   - A command-line-based operating system.  
   - Features: Simple interface, file management.  

(iii) **MS Word Features:**  
   - Text editing and formatting tools.  
   - Limitations: Limited spreadsheet capabilities.  

**MS Excel Features:**  
   - Data management and calculation tools.  
   - Limitations: Not suitable for advanced document editing.  

**Detailed Explanation (Hinglish):**  
- **Registers:** Yeh CPU ke andar hoti hain aur high-speed memory provide karti hain.  
- **DOS:** Ek command-based OS jo file aur system ko manage karta hai.  
- **MS Word aur MS Excel:** Word text formatting ke liye aur Excel calculations ke liye use hota hai.

### **Question 4:**  
**Differentiate between primary and secondary memory. Also discuss cache memory and system buses.**  

**Answer:**  
- **Primary Memory:**  
  - Faster and directly accessible by the CPU.  
  - Examples: RAM, ROM.  

- **Secondary Memory:**  
  - Slower and used for permanent storage.  
  - Examples: Hard Disk, USB.  

**Cache Memory:** A small, high-speed memory used to store frequently accessed data.  

**System Buses:**  
  - **Data Bus:** Transfers data.  
  - **Address Bus:** Specifies memory locations.  
  - **Control Bus:** Controls signals between CPU and peripherals.  

**Detailed Explanation (Hinglish):**  
- **Primary Memory:** Fast hoti hai aur CPU directly access karta hai.  
- **Secondary Memory:** Slow hoti hai aur permanent storage ke liye hoti hai.  
- **Cache Memory:** Frequently used data ko store karta hai.  
- **System Buses:** Data aur instructions ko transfer karne ke liye buses ka use hota hai.

### **Question 5:**  
**Differentiate application and system software. Give examples. Also explain 1’s and 2’s complement and give two examples of each.**  

**Answer:**  
**Application Software:** Used for specific tasks. Examples: MS Word, VLC.  
**System Software:** Manages hardware and software. Examples: Operating Systems.  

**1’s Complement:** Invert all bits of a binary number. Example: 1010 → 0101.  
**2’s Complement:** Add 1 to the 1’s complement. Example: 0101 + 1 = 0110.  

**Detailed Explanation (Hinglish):**  
- **Software Types:** Application software specific kaam ke liye hota hai, aur system software system ko manage karta hai.  
- **1’s Complement:** Binary digits ko reverse karte hain. Example: 101 → 010.  
- **2’s Complement:** 1’s complement mein 1 add karte hain.

### **Question 6:**  
**Explain the term 'Batch processing.' Give an example. How is a Multiprocessing OS different from a Multiprogramming OS?**  

**Answer:**  
**Batch Processing:** Collecting data/jobs and processing them in batches without user interaction. Example: Payroll system.  

**Difference:**  
- **Multiprocessing OS:** Uses multiple processors to execute tasks simultaneously.  
- **Multiprogramming OS:** Manages multiple tasks but with a single CPU.  

**Detailed Explanation (Hinglish):**  
- **Batch Processing:** Data ko groups mein process karna. Example: Salary calculation.  
- **Multiprocessing aur Multiprogramming OS:** Multiprocessing mein multiple CPUs hote hain; multiprogramming mein ek CPU multiple tasks handle karta hai.
